# Positioning

AI NFT Forge is a self-hosted, white-label AI and NFT product platform for agencies, creator teams, portrait or event studios, and NFT-native studios that need branded collectible experiences rather than a generic photo filter tool.

The product direction is premium and operationally serious:

- B2B white-label first
- Premium storefront experience as a core differentiator
- Self-hostability as a core buyer requirement
- Support for bespoke 1/1 items and edition drops over time

It should be positioned as infrastructure for branded collectible launches:

- ingest client photos
- generate collectible-style variants
- curate final assets
- publish branded collection pages
- mint onchain later

The repository is still in foundation state. Current work should strengthen architecture, delivery flow, and self-hostable operations before feature depth.
