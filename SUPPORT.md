# Support

## Scope

This repository is an actively developed product codebase, not a hosted service.

Support is best-effort and depends on maintainer availability.

## Good Support Requests

Useful support requests include:

- the exact command that failed
- the relevant package or app
- whether you were using local development or `infra/docker/docker-compose.selfhost.yml`
- the observed error output
- recent changes or branch context
- steps already attempted

## Not Supported Here

- posting secrets or private credentials
- requesting unsafe shortcuts around security controls
- bypassing documented architecture and contribution rules
- using issues for confidential vulnerability disclosure

Use [SECURITY.md](/Users/mc/development/blockchain/ethereum/ai-nft-forge/SECURITY.md) for vulnerability reporting.

## Collaboration Boundary

For code contributions, design changes, or documentation updates, follow [CONTRIBUTING.md](/Users/mc/development/blockchain/ethereum/ai-nft-forge/CONTRIBUTING.md).
