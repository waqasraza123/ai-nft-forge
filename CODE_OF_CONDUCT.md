# Code of Conduct

## Our Standard

This repository is expected to be a professional, respectful collaboration space.

Contributors are expected to:

- act in good faith
- communicate clearly and respectfully
- accept technical critique without hostility
- avoid harassment, discrimination, intimidation, and personal attacks
- keep discussions focused on the work, the users, and the repository

Unacceptable behavior includes:

- abusive or demeaning language
- harassment or threats
- trolling, deliberate disruption, or bad-faith review behavior
- publishing private information without consent
- discriminatory behavior based on personal identity or background

## Scope

This code of conduct applies to:

- issues
- pull requests
- code review discussions
- repository documentation
- any project-managed collaboration channel tied to this repository

## Enforcement

Project maintainers may remove, edit, or reject contributions or interactions that violate this code of conduct.

Repeated or severe violations may lead to temporary or permanent loss of participation rights in the repository.
